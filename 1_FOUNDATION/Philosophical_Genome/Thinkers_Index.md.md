## 1. Overview

The **Thinkers Index** consolidates all primary and secondary philosophical sources that inform Codex OS.  
It functions as a **conceptual reference system**, ensuring that every creative, analytical, or narrative decision is anchored in clear intellectual lineage.

Each thinker listed below connects to:
- A *Core Principle* in `/Philosophical_Genome/Core_Principles.md`
- A *Behavioral Manifestation* in system protocols
- A *Tone Calibration Reference* in `/4_REFLECTION/Echo_Layer/`

## 2. Structure

| Field | Description |
|--------|-------------|
| **Thinker** | The philosopher, creator, or strategist who defines part of the Codex mindset. |
| **Core Concept** | The key idea that informs Codex behavior. |
| **Main Influence Area** | Which part of the system it shapes (Foundation, Awareness, Reflection). |
| **Reference Work** | Book, essay, or body of work. |
